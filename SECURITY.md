# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ARC UI, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email **hello@arclight.build** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This policy applies to the ARC UI packages published under `@arclux/` on npm:

- `@arclux/arc-ui` (web components)
- `@arclux/arc-ui-react`
- `@arclux/arc-ui-vue`
- `@arclux/arc-ui-svelte`
- `@arclux/arc-ui-angular`
- `@arclux/arc-ui-solid`
- `@arclux/arc-ui-preact`

## Supported Versions

Only the latest released version is actively supported with security updates.
